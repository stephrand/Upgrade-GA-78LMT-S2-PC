# Upgrade Process Notes – GA-78LMT-S2 PC

## Step 1 – Preparing the PC
- Powered off PC and unplugged all cables
- Opened case

## Step 2 – RAM Upgrade
- Removed old ADATA RAM modules
- Installed 2 × 8GB G.SKILL RipjawsX modules
- Verified correct seating

## Step 3 – Storage Upgrade
- Removed WD Blue 1TB HDD
- Installed Crucial BX500 1TB SSD
- Checked SATA connections

## Step 4 – Close Case Before Boot
- Closed the PC case before powering on the system for the first time after upgrades
- Ensured all cables were secure

## Step 5 – BIOS Configuration
- Powered on PC and entered BIOS
- Verified new RAM and SSDs detected
- Set boot order to USB

## Step 6 – Windows Installation
- Created Windows 10 Home USB with Media Creation Tool
- Performed clean installation of Windows 10 Home
- Installed necessary drivers (chipset, GPU, network)

## Step 7 – Final Steps
- Initialized new SSD in Windows Disk Management
- Verified total storage and system performance
- Confirmed stable operation of RAM and SSDs
